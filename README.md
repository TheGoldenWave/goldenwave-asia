# GoldenWave Asia

Personal website for 金波 / Jason Zhang — AI products, Agent workflow systems, knowledge assets, and GoldenWave motion identity.

## Preview

Serve locally:

```bash
python3 -m http.server 5173
```

Then visit <http://localhost:5173>.

Opening `index.html` directly can work for the static shell, but the local server path is preferred because CSS, JS, and locale assets are loaded as separate files.

## Design direction

Noir-Gold editorial portfolio based on `DESIGN.md`:

- Deep charcoal / black surfaces
- Gold GoldenWave motion motif with knowledge waveform / system pulse visuals
- Noto Serif SC display typography
- Hanken Grotesk labels/body
- Editorial portfolio layout with a Personal Agent Harness system map
- GSAP scroll motion, Canvas background waves, and responsive reduced-motion fallbacks
