# GoldenWave Asia

Personal website for 金波 / Jason Zhang — AI products, workflow systems, knowledge assets, and GoldenWave motion identity.

## Preview

Open `index.html` directly in a browser, or serve locally:

```bash
python3 -m http.server 5173
```

Then visit <http://localhost:5173>.

## Design direction

Noir-Gold editorial portfolio based on `DESIGN.md`:

- Deep charcoal / black surfaces
- Gold GoldenWave motion motif
- Noto Serif SC display typography
- Hanken Grotesk labels/body
- Minimal, glassmorphic, quiet-luxury layout
